November 2021
Created an octopus-design ultrasonic sweeping sensor with accompanying output application.

The Task: Create an ultrasonic sensor device which adhered to the following requirements:
...
Requirements for the design:
• Uses an L-bracket for mounting the ultrasonic sensor
• Ultrasonic sensor is mounted in a way to allow sweeping motion back and forth.
• Functionality includes scanning distance to surrounding objects at intervals.
• Functionality includes graphically outputting distance to surrounding objects in "radar-like" format using Processing.
• Some level of "wow factor" included for maximum grade.

The professor did not specify visual requirements, so I made it into a light-up octopus with eyebrows.
I not only included all required functionality, but also added mounted servos to the eyebrows.

When the sensor scans a distance to an object of over 10 feet, the eyebrows tilt down (as it is "concentrating" to see the object. When it scans a distance to an object of under 10 feet, the eyebrows tilt back up (as it can see the object clearly).

Professor feedback: “[...] probably the most impressively completed A2 in the history of ISTA 303.”

November 2020
Completed some last-minute artwork for the Smart Thermostat configuration of the AWS IoT EduKit.

My dad's team was in the final stretch of developing the EduKit's Smart Thermostat module, and ... they found themself without actual UI artwork. It was too late to request Amazon official art, and my dad had seen me make UI mockups and artwork in the past, so I was asked if I was interested in helping.
My last final project was due in a few days, but I figured I had some extra time in-between test runs and said yes.

Requirements for the design:
• It needed to use Amazon official colors, RGB values were provided.
• It needed to include some color-changing portion to show the current color of the LEDs attached to the device.
• It needed to have a day and a night variant.
• There needed to be some GIF icon to show whether or not the device fan was running, and an ON/OFF icon.
• There needed to be some thermometer icon and temperature actual value location.

I ended up going through two or three different house designs and a lightbulb option before we found one we chose. The fan went through two iterations, one with thicker lines and one with thinner (which ended up being chosen). We also had the decision to make the fan ON/OFF either a switch or a button. A switch and a speed slider were chosen to save room on the screen.
